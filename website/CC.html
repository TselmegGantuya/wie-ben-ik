Clean Code

Waarom wil ik Clean Code leren?

Clean Coding is netjes werken, zorgen dat je code makelijk te lezen is en makelijk te veranderen. Als je goed om gaat met Clean Code kan je veel beter samenwerken. Ik vind het ook belangrijk om netjes te werken zodat ik veel meer mijn fouten kan veranderen. Door het makelijker te laten lezen zorg je er voor dat meer mensen je code kunt gebruiken. Door het makelijker te laten lezen zorg er voor dat je goed kan dingen veranderen en ook laten vernieuwen.
