Java

Wat is Java?

Java is een objectgeorienteerde programmeertaal. Java is een platformonafhankelijke taal die qua syntaxis grotendeels gebaseerd is op de (eveneens objectgeorienteerde) programmeertaal C++. Java beschikt echter over een uitgebreidere klassenbibliotheek dan C++.Java ontstond begin jaren 90 in een klein dochterbedrijf van Sun Microsystems onder leiding van James Gosling.Over de nummering van de versies is veel te doen geweest. Na het uitbrengen van versie 1.2 is de 2 vaak vermeld achter Java, wat dan een hele reeks boeken opleverde die "Java 2 behandelden", ook als het om versie 1.3 of 1.4 ging. Vanaf versie 1.5.0 was er een dubbele nummering: had men het over Java 5.0, dan ging het over de JRE of Java Runtime Environment. Het versienummer 1.5.0 verwees heel specifiek naar de JSDK of Java Software Development Kit, met andere woorden de ontwikkelomgeving. SE staat overigens voor "Standard Edition".
